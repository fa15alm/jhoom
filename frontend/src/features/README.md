# Features

Feature folders hold code that belongs to one product area.

Suggested feature folders as the backend is connected:

- `auth/` - login, signup, session helpers.
- `health-plan/` - onboarding answers, generated plan state, AI plan helpers.
- `logs/` - workout, nutrition, sleep, steps, and calories logging.
- `dashboard/` - dashboard view models built from logs and integrations.
- `social/` - posts, friends, comments, and privacy controls.
- `milestones/` - goals, milestone progress, and completion logic.
